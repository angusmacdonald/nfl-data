Three Years of YAC

The following spie charts show receiving yards per wide receiver, with YAC indicated by the outer slice on each segment. The width of each segment is based on the relative number of receptions, the length of each segment is based on the relative number of receiving yards.

Notes

It's hard to visualize YAC for running backs, because the stats produced by various outlets always shows a much higher YAC than the number of receptions. I'm not entirely sure why this is, but it means that this visualization shows all running back yards as consisting entirely of YAC.
